# Student Grade Management System

## Description
A simple Python project to manage student grades using SQLite.

## Technologies
- Python
- SQLite

## Features
- Add new students
- Record grades for subjects
- View all grades of a student
- Calculate average grade automatically

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/kurseasy/student-grade-system
   cd student-grade-system
2. Run the program:
   python grades.py
